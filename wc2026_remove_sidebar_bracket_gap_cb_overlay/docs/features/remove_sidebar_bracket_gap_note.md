# Remove Sidebar-Bracket Gap Note

The latest visual target removes the large horizontal gap between:

```text
Teams sidebar
```

and

```text
Bracket surface
```

The desired layout is a tight split-screen:

```text
[ Teams ] [ Bracket ]
```

not:

```text
[ Teams ]        [ Bracket ]
```

This supports the current compact UI direction:

- narrow team well
- compact flag-only bracket slots
- bracket starts immediately beside the team well
