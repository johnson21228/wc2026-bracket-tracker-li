# Round-of-32 Source Pair Band Rule

## Purpose

The bracket should visually group each Round-of-32 match pair so users can see which two teams form a source unit for later-round advancement.

## Rule

Each Round-of-32 match should appear inside a subtle horizontal band.

The band should communicate:

```text
these two teams form one R32 match
the winner feeds the next round
```

## UI guidance

Bands should be subtle, not visually heavy.

They may use:

- light background tint
- soft border
- small match label
- native tooltip

## Screenshot rule

The bands should improve screenshot readability so a future reviewer or II reasoner can identify R32 pairings more reliably.
