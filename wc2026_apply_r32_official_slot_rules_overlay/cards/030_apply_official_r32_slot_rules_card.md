# Card 030 — Apply Official R32 Slot Rules

## Intent

Add official Round-of-32 slot formulas to the site and data model.

## Acceptance

- `data/official_round_of_32_slot_rules.json` contains the slot formulas.
- Empty R32 slots show compact labels such as `2A`, `1E`, or `3 A/B/C/D/F`.
- Filled R32 slots show flag-only display.
- Slot rule is preserved in title and data.
- Drag/drop remains R32-only.
- New static HTML release is created.
