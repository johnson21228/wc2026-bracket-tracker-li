# Official Round-of-32 Slot Rules Rule

## Purpose

Round-of-32 drop targets should show the qualification rule for that position.

Examples:

```text
2A
1E
3 A/B/C/D/F
```

## Authority

Use the official FIFA Round-of-32 match schedule and tournament regulations.

The fixed slot formulas define:

- group winners
- group runners-up
- candidate pools for best third-place teams

The exact best-third-place assignments depend on which eight third-place teams qualify.

## UI rule

Each empty R32 drop target should show a compact slot label.

When a team is dropped, the flag may replace the visible rule label, but the slot rule should remain preserved in data and title/tooltip.
