# Create SPINE and README

Create a first `SPINE.md` and `README.md` for this Workbench LI repo.

The SPINE should define:

- purpose
- governing idea
- boundary between LLM reasoning and human custody
- source of authority
- first useful outcome

The README should invite practical use:

- what this repo is
- who it is for
- how to start
- what to upload to an LLM
- how to generate history and pack
- how the repo grows through use

Keep both documents plain, direct, and adoption-friendly.
