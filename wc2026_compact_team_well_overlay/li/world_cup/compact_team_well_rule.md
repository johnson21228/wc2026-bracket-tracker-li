# Compact Team Well Rule

## Purpose

The 48-team source well should be compact so users can see more teams without excessive scrolling.

## Rule

Team well items should be roughly half the previous width.

The item may show:

- flag
- short team name

The item should hide secondary identifiers in the public view.

Full identity should remain available in tooltip/title and state data.

## Data preservation

Do not remove:

- team name
- abbreviation
- group
- canonical data

Only compact the visual display.
