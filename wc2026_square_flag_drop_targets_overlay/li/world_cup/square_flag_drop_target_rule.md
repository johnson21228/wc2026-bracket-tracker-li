# Square Flag Drop Target Rule

## Purpose

The bracket should use compact square slots.

The slot should visually show only the team flag. The full team identity should be available by hover tooltip and preserved in data.

## Rule

Bracket display:

```text
🇲🇽
```

Tooltip:

```text
Mexico (MEX) - Group A
```

## Drag/drop rule

Only Round-of-32 slots are drop targets.

Later-round slots should remain compact passive advancement slots until winner selection/result logic is implemented.

## Data preservation

Do not remove team names, abbreviations, groups, or state from the data model.

This is a visual compacting rule only.
