# Bracket Spacing Tuner Note

The current bracket layout may need visual tuning.

The Stage 0 approach is to add sliders for key CSS variables:

```text
--r16-top
--r16-gap
--qf-top
--qf-gap
--sf-top
--sf-gap
--final-top
```

The user can open the site, adjust the sliders, copy the chosen CSS values, and provide those values back for Capture Back.

This is preferable to guessing.
