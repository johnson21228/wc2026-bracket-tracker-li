# Capture Back macOS Review Surface Visual

## Purpose

This folder stores the visual assets for explaining the macOS Capture Back workflow.

## Accepted generated infographic

- `capture_back_on_macos_review_surface.jpeg`

This is the accepted generated image for the current template-level Capture Back review-surface protocol.

It teaches:

```text
Download to Downloads.
Apply from repo root.
Paste stdout back to chat.
Review opened files.
Commit only after approval.
```

## Source reference

- `source_stage0_original_workflow_reference.jpeg`

The source reference visual captured the early Workbench loop where one ChatGPT conversation could play both the Workbench role and the II role, while Terminal executed the durable Capture Back update.

The generated infographic updates that older pattern into a practical macOS Capture Back operating diagram.

## Updated framing

Use:

```text
Capture Back on macOS: From Chat Reasoning to Workbench Memory
```

Do not lead with `Stage 0` language when teaching the current Capture Back protocol.

## Why this belongs in the template

Every generated Workbench that uses the current macOS workflow can inherit this operating picture. It teaches the bridge between chat reasoning and local repo memory.
