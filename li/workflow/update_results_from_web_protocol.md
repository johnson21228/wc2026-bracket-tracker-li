# UPDATE RESULTS Protocol

When the user says `UPDATE RESULTS`, retrieve and follow `prompts/update_results_from_web.md`.

This protocol protects live tournament data from being frozen prematurely. A result can be patched only when a reliable source clearly indicates `Final`, `FT`, `Full-time`, `Complete`, or an equivalent completed-match recap.

Live or in-progress matches must be classified as `WATCH`, not patched as final.
