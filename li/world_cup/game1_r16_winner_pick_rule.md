# Game 1 R16 Winner Pick Rule

Game 1 may support a first knockout continuation phase while preserving its original R32 assignment purpose.

A Round of 16 winner pick is valid only when both source Round of 32 slots for that R16 match contain assigned teams.

The R16 slot geometry must come from the uniform SVG gameboard manifest. Runtime code must not maintain independent R16 position constants.

R16 winner picks must be stored separately from R32 assignment picks so later officialization can preserve Game 1 scoring evidence while allowing the bracket board to continue into the Game 2/official-truth lifecycle.

This rule does not authorize QF/SF/Final Four picking yet. Those rounds should be added through separate cards after R16 behavior is verified.
