# Final Four Center-Stack Geometry Rule

Final Four center-stack geometry must originate in the source-truth SVG and be projected into the runtime manifests.

Required slots:

- `FINAL-LEFT`
- `CHAMPION`
- `FINAL-RIGHT`

`CHAMPION` is intentionally twice as wide and twice as tall as the upper/lower semifinal-winner cells.

`CENTER-FINAL-FOUR` must not return as independent runtime pick-slot geometry.
