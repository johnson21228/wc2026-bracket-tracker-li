# Game 2 Bracket Item Seed Rule

Game 2 begins by creating 32 Round-of-32 bracket items from a fixed seed.

A bracket item is the movable game object that carries one team through the knockout board. It has an id, round, source slot, board index, seed index, team payload, and status.

Game 2 must not use Game 1 picks as bracket truth. Game 1 may later be compared as a tiebreaker only.
