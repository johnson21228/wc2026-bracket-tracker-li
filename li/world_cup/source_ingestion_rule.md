# Source Ingestion Rule

## Purpose

The Workbench can accept source artifacts before formal structured data exists.

A source artifact may be:

- image
- screenshot
- PDF
- webpage
- exported JSON
- static HTML release
- manual note

## Preservation rule

Preserve original source artifacts whenever possible.

## Derived data rule

Any derived data must record:

- source artifact path
- derivation date
- authority status
- known uncertainties

## Conflict rule

If a higher-authority source later conflicts with a lower-authority source, record the correction instead of silently replacing history.
