# Pixelated country flag background layer rule

The World Cup Game 1 visual background may use a selected country flag from the page team data as a decorative background layer.

Rules:

- The flag background must be behind all gameplay controls and hit targets.
- It must be generated from existing page team/flag data, not from a separate untracked image dependency.
- It must use low opacity and a wash layer so the bracket remains legible.
- It must preserve pointer behavior; the decorative layer must not capture events.
- It must be selectable but not required for gameplay.
