# WC2026 Lifecycle Background Softening Rule

The Bracketeering Hub lifecycle-stage background images are presentation-only.

Both Group Stage and Knockout Stage backgrounds should share the same softened visual treatment so the bracket cells, pick menus, group panels, and text remain readable.

This rule must not change:
- lifecycle-stage image selection
- lifecycle-stage naming
- pick availability
- pick menu activation
- preselection
- bracket state
- storage
- scoring

The preferred implementation is CSS-only through the shared board background layer.
