# Info panel copy rule

The player-facing surface opened by the upper-left `i` button is the Info panel, not the Rules panel.

The copy should use “World Cup Bracketeering Hub” and avoid implying official FIFA affiliation. The panel must explain how to play, map-style navigation, local browser pick storage, Group Stage, Knockout Stage, lock timing, Group Stage tiebreaker role, and scoring.

Legacy `data-rules-panel-*` hooks may remain as aliases during migration, but new runtime naming should prefer `setupInfoPanel` and `data-info-panel-*`.
