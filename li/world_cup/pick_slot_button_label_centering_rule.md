# Pick Slot Button Label Centering Rule

Pick-slot button labels and values must remain visually centered inside the slot button frame.

This rule is presentation-only.

It must not:

- create pick state
- create preselect state
- change Game 1 / Game 2 state ownership
- restore generic R16+ hover/preselect highlighting
- alter pick write behavior

The runtime CSS must explicitly center pick-slot labels and values.
