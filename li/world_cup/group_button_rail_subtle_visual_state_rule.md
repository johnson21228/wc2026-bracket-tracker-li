# Group Button Rail Subtle Visual State Rule

The group button rail must be present but visually quiet by default.

Each group tile renders as a partially translucent button while at rest. The tile must remain readable and discoverable, but it should not compete with bracket picks, slot labels, or the board artwork.

When the user hovers, focuses, touches, or tracks over a group tile, that tile becomes fully opaque and may increase contrast, border strength, shadow, or saturation.

The active visual state must not change tile size, group order, 2×2 flag grid layout, rail position, pick state, or group panel state.

This rule is limited to visual emphasis. It does not define group panel anchoring or placement.
