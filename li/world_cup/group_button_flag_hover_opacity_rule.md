# Group Button Flag Hover Opacity Rule

Group rail flags are decorative evidence inside a single group tile button. They are not individual controls.

By default, group rail flags should be slightly translucent so the rail remains subtle against the bracket board.

On group tile hover, keyboard focus, active press, or active tracking-over state, the flags inside that tile must become fully opaque.

The opacity change must not move flags, change the 2×2 grid, alter group order, or mutate pick state.
