# Current Group Result Evidence Rule

Current group-stage result patches must start from repo data and update only matches with reliable final-result evidence.

A match may be patched as final only when an external source clearly indicates Final, FT, Full-time, Complete, or equivalent completed-match wording.

Live or in-progress matches must be classified as WATCH and left unmodified until final evidence exists.

Source evidence belongs under `source/text/` and runtime data remains derived from that evidence. Standings may be updated only from confirmed final inputs.
