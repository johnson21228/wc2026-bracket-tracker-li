# Game 1 QF/SF Winner Pick Rule

Game 1 advancement picks beyond R16 are resolved from prior-round winner state, not from group eligibility.

A QF slot is available only when both feeding R16 slots have winner picks. The QF choice menu must contain exactly those two R16 winners.

An SF slot is available only when both feeding QF slots have winner picks. The SF choice menu must contain exactly those two QF winners.

QF/SF pick state must remain separate from R32 assignment picks. It may also remain separate from the first R16 winner-pick storage during this incremental migration.

This rule intentionally stops at SF. Final/Champion behavior should be added only after R16/QF/SF source-slot behavior is verified.
