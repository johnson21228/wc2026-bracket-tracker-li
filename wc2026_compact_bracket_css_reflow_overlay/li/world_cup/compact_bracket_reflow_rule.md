# Compact Bracket Reflow Rule

## Purpose

When bracket slots are compact square flag tiles, the surrounding bracket layout must also shrink.

## Rule

Do not only shrink the slot. Also shrink and retune:

- R32 match bands
- connector line lengths
- round column widths
- later-round slot widths
- trophy/final spacing
- spacing tuner defaults

## Reason

If only the slot shrinks, the old wide bands/connectors remain visually disconnected from the new compact item.

The full bracket layout must be reflowed around the compact slot.
