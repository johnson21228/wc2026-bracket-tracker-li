Game 1 — Pick the Round of 32 field

Before the final round of group-stage matches begins, choose the teams you think will make the Round of 32.

Game 1 lock time

All Game 1 picks must be submitted before kickoff of the earliest third-round group-stage match.

Some Round-of-32 places may already be known by then. That is part of the game. Many other places will still depend on final group-stage match results, including standings, goal difference, and third-place qualification.

Scoring

  • 1 point for each correctly predicted Round-of-32 team.
  • No perfect-32 bonus is currently set.

Game 2 — Pick the knockout bracket

After the knockout bracket is available, pick winners through the bracket.

Scoring

  • Round of 32 correct winner: 1 point.
  • Round of 16 correct winner: 2 points.
  • Quarterfinal correct winner: 4 points.
  • Semifinal correct winner: 8 points.
  • Champion: 16 points.

Game 2 tiebreakers

  1. Higher Game 1 score. The tied player with the better Game 1 score wins the tiebreaker.
  2. Third-place pick. If still tied, the tiebreaker is the player's third-place pick.
  3. Earliest valid Game 1 save timestamp. If still tied, the player whose final Game 1 save was earliest wins, as long as that save happened before the Game 1 lock time.

Rule status

These scoring rules are proposed until the pool owner locks them. After lock, changes should be captured as scoring-rule changes.

Developer note

Game selector is currently UI-only. Choosing Game 2 updates only the selector state and the rules text shown here. It does not switch gameplay, scoring, storage, routes, Supabase state, data loading, or board rendering yet.
