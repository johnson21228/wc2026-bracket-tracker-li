# Card 169 — Copyable developer properties

## Intent

Add a developer control that shows all live board presentation properties in a copyable form.

## Use

Tune the developer controls, copy the JSON snapshot, and paste it into the chat. The pasted snapshot can then define promoted site defaults.

## Acceptance

- Developer frame includes a copyable properties panel.
- Properties update when controls change.
- Copy works through a button when browser permissions allow.
- Text remains selectable for manual copy.
- Source SVG and geometry assets are not mutated.
