# Card 290: iPhone Touch Map Mode

## Intent

Make the Bracketeering board behave like a map on iPhone Safari while preserving desktop behavior.

## Acceptance

- One-finger touch drag pans the board viewport.
- Two-finger pinch zooms the board around the pinch midpoint.
- Fixed + / − controls remain viewport/safe-area chrome.
- Interactive controls remain tappable.
- Desktop mouse drag, double-click zoom, and wheel zoom remain unchanged.
- `make verify` passes.
