# Capture Back: Player Standings No Pick Links

## Outcome

Player Standings no longer exposes a link or button to open another player's picks.

## Behavior

- Player Standings still lists joined/public players.
- Player Standings still reads storage-backed standings rows.
- Player rows show public player names as plain, non-interactive standings text.
- The prior `View picks` board-viewer affordance is intentionally superseded.
- No save/write path is added.
