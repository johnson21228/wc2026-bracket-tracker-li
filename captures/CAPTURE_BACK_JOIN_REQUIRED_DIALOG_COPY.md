# CAPTURE BACK: Join Required Dialog Copy

## Intent

The Join/Profile dialog must clearly communicate that Bracketeering requires joining the Pool before play. The old exploratory/local-play posture is no longer player-facing.

## Signed-out dialog copy

Title:

Join the Pool

Intro:

Playing Bracketeering requires you to join the Pool.

Guidance note:

Use Google sign-in to avoid email verification. If you use email, check your spam folder if the sign-in link does not appear in your inbox.

State:

Not joined yet.

## Signed-in dialog copy

When the player is signed in, the same dialog becomes the Profile dialog.

Signed-in players only need support to edit their player name and log out.

Signed-in copy:

Profile

Edit your player name or log out.

Edit your player name below, or log out.

Joined status: Joined

## Boundary

Do not tell signed-out players they can keep exploring, keep playing locally, or that no account is needed for local play.

Do not use the signed-in Profile dialog for saved-picks status. Saved-picks status belongs to the joined-picks status surface, not the Profile dialog.
