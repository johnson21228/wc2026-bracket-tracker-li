# Force Adjacent Panes Rule

## Purpose

The left team pane and right bracket pane should sit directly next to each other.

## Rule

The page-level layout should use a tight two-column structure:

```text
300px team pane | 8px gutter | bracket pane
```

Avoid large blank horizontal gaps between the panes.

## Implementation note

This must be controlled at the main container level, not only inside the bracket.
