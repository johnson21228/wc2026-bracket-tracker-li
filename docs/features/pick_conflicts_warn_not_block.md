# Pick conflicts warn, not block

The bracket tracker separates source-scoped menu eligibility from bracket-state validity.

The menu keeps the slot's source boundary. Canada should appear in Group B-derived slots, but not in unrelated source groups. If Canada is already picked in another Round-of-32 slot, choosing Canada again is a conflict, not a reason to block the pick.

The pick is preserved and the cell is rendered with the invalid-pick warning treatment: thin red outline plus red `!` marker.

This keeps user intent visible while making bracket conflicts obvious enough for the user to resolve.
