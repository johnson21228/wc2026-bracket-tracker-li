# Final Four Center-Stack Pick Menu

The Final Four center-stack cells are interactive pick surfaces.

## Cells

- `FINAL-LEFT`
- `CHAMPION`
- `FINAL-RIGHT`

Clicking one of these cells preselects the matching canonical pick slot and opens the existing pick menu. Choices are generated by the model from precedent constraints.

## Constraints

- `FINAL-LEFT`: winner/candidate from `L-SF-01` and `L-SF-02`
- `FINAL-RIGHT`: winner/candidate from `R-SF-01` and `R-SF-02`
- `CHAMPION`: selected `FINAL-LEFT` and selected `FINAL-RIGHT`
- `THIRD-PLACE-WINNER`: semifinal losers

The implementation must not create a second pick menu system and must not restore lifecycle-stage pick gating.
