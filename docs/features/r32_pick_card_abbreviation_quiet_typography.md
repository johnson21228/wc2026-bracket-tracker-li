# R32 Pick Card Abbreviation Quiet Typography

This repair tones down the three-letter team abbreviation shown on filled R32 pick cards.

The prior rendering correctly changed the compact card from full team name to team abbreviation, but the letters were still too visually heavy. The card read like a poster headline instead of a quiet bracket-board label.

The target is:

- flag remains the primary visual identifier
- abbreviation remains legible
- abbreviation is regular/medium weight, not bold
- the card fits the board-defined slot
- tooltip/details continues to carry full team name and explanation
