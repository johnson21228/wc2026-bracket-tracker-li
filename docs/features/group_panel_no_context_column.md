# Group panel without player-facing Context column

The group standings panel renders the core standings columns only:

- Rank
- Team
- MP
- W
- D
- L
- GF
- GA
- GD
- Pts

The underlying standings model may still carry qualification context for internal use, but the player-facing table does not render context badges such as `Group Winner`, `Runner Up`, `Third Place Candidate`, or `Fourth Place`.

Completed match evidence and highlight links remain below the standings table.
