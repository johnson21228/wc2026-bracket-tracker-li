# Game 2 Foundation Layers

Game 2 starts from the same visual layer contract as Game 1.

- Back layer: pub image atmosphere.
- Middle layer: transparent 32-slot bracket geometry PNG.
- Picking layer: intentionally deferred.
- Rules layer: intentionally deferred.

This keeps Game 2 visually aligned with Game 1 while leaving the game objective and pick constraints open for the next Capture Back.
