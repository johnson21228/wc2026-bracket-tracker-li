# Game 2 Bracket Item Seed

This feature seeds Game 2 by rendering 32 visible bracket items on the shared gameboard.

The seed is available at `site/data/game2_round_of_32_seed.json` and embedded in `site/game2/index.html` so the prototype works from `file://`.

Each bracket item is rendered as a visible card containing flag, country, slot, and bracket item id.
