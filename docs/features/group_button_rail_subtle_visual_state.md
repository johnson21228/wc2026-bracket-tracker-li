# Group Button Rail Subtle Visual State

The group button rail is a secondary inspection surface. It gives quick access to group panels without becoming the dominant visual object on the gameboard.

At rest, each group tile should be partially translucent. The group label and four flags remain readable, but the rail should feel like a quiet affordance.

When the user hovers, focuses, touches, or tracks over a group tile, the tile becomes fully opaque and may gain stronger contrast, border, shadow, or saturation. This active state communicates interactivity without changing the tile's size, order, rail position, or 2×2 flag grid layout.

This feature is CSS-only. It does not change panel placement or event routing.
