# Pick Menu Group Panel Link

The pick menu is not only a selection surface. It is also the user’s local explanation of why the available choices belong in the current bracket slot.

When choices come from World Cup groups, the menu should keep the group visible. Teams should be collected under group labels, and each group label should open the shared group standings panel for that group.

This supports the Game 1 flow:

- User taps a bracket slot.
- Pick menu opens.
- Menu shows choices collected by group.
- User can tap `Group A`, `Group C`, etc. to see the standings panel for that group.
- User returns to the pick context without losing the bracket flow.

For third-place source slots, the grouped display is especially important because a label like `3RD A/E/H/I/J` is not a single team source until the third-place allocation has been resolved. The menu should preserve those possible source groups and let the user inspect each group’s standings.
