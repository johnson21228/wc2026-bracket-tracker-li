# Game 1 R16 Live Candidate Resolution Repair

R16 slots must build their candidate menu from the current upstream R32 picks.

For `L-R16-01`, the intended feeders are:

    L-R32-01
    L-R32-02

The rendered R32 pick cards may use manifest IDs instead:

    R32-L-M1A
    R32-L-M1B

The repair adds alias resolution so the R16 menu can find the live teams regardless of which valid slot identifier is present in the pick store.
