# Pick Slot Button Label Centering

Pick-slot buttons are compact visual targets. Their label/value text must be centered in the visible button frame.

The centering rule applies to:

- empty pick prompts
- selected pick values
- compact picked-cell identity text
- later-round winner labels

This is a CSS/rendering contract. It does not change selection state, preselect state, pickability, or Game 1/Game 2 ownership.
