# Group Button Flag Hover Opacity

The group button rail is a secondary inspection surface. Its flag grid should be readable but not visually dominant while idle.

Flags inside each group button should be slightly translucent by default. When the user hovers over, focuses, presses, or tracks over the group button, the flags in that button become fully opaque. This reinforces that the whole group tile is interactive without turning each flag into its own button.

This feature does not change group panel placement, pick menu behavior, or model data.
