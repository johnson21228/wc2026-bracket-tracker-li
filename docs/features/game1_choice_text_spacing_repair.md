# Game 1 Choice Text Spacing Repair

This repair fixes chooser options that visually collapsed the team name into the abbreviation, for example `MexicoMEX · Group A`.

The choice row now has a dedicated text wrapper with an explicit gap between team name, abbreviation, and group metadata. This is a readability fix only; it does not change R32 slot rules, pointer resolution, manifest geometry, pick storage, or menu choices.
