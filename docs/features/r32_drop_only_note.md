# Round-of-32 Drop Only Note

The current drag/drop behavior should be limited.

Users should drag teams only into Round-of-32 slots.

Later rounds should be filled by bracket logic:

```text
R32 placement
↓
winner selection
↓
R16
↓
winner selection
↓
QF
↓
...
```

This avoids invalid bracket states where a team is dropped directly into a semifinal or champion slot.
