# Knockout background deemphasis

The knockout pub background remains the accepted runtime asset:

`site/assets/board/knockout_pub_background.jpeg`

The visual deemphasis is presentation-only. The JPEG is not regenerated or modified for opacity. Runtime deemphasis is applied through `.board-background-layer` opacity in `site/css/board.css`, so the image can remain an authoritative visual asset while the board/pick UI stays legible.
