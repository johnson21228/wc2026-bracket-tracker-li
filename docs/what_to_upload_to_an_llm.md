# What to Upload to an LLM

For a full review or improvement pass, upload the latest pack:

```text
dist/*.pack.zip
```

For a short interpretation of repo evolution, upload only the latest history artifact:

```text
outputs/history/repo_history_for_llm_*.md
```

For a focused source review, upload only the specific source file or folder that governs the question.

Avoid uploading everything when a narrower artifact is enough.
