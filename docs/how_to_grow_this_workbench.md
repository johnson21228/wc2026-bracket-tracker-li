# How to Grow This Workbench

Grow the workbench through small, durable changes.

Good growth usually adds one of these:

- a source document that governs the work
- a prompt that can be reused
- a continuity card that records intent, reasoning, validation, and handoff
- a generated artifact worth preserving
- a validation check
- a repo-history interpretation
- a README or SPINE clarification

Prefer narrow overlays over broad rewrites.
