# First Use

This template is meant to help you create a new Workbench LI repo.

Start by opening this prompt in your LLM reasoning tool:

```text
prompts/00_create_a_workbench_li_repo.md
```

Describe the domain, project, person, team, or problem you want the workbench to support.

The first goal is not completeness. The first goal is to create a durable place where the work can start accumulating.
