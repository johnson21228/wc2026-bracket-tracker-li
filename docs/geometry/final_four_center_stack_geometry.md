# Final Four Center-Stack Geometry

The Final Four center grouping is represented by explicit source-derived geometry slots.

## Source truth

`site/assets/playfield/uniform_pick_card_gameboard.svg`

## Slots

- `FINAL-LEFT`
- `CHAMPION`
- `FINAL-RIGHT`

The `CHAMPION` cell is twice the width and twice the height of the upper and lower semifinal-winner cells.

`CENTER-FINAL-FOUR` is legacy visual-only terminology and must not be used as runtime pick-slot geometry.
