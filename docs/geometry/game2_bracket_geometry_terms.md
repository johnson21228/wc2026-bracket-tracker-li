# Game 2 Bracket Geometry Terms

## Board asset

The image or SVG that communicates the visible bracket layout.

## Bracket slot

A rectangular or shaped logical placement region on the board. It can receive a bracket item and may participate in advancement topology.

## Bracket item

The visible object placed into a bracket slot. For R32 seed slots this is usually a team. Later it can be a user pick or official winner.

## Advancement edge

A directed relation from one or more source slots to a later slot. For example, two R32 slots feed one R16 slot.

## Captured-from-image geometry

A geometry model manually measured or approximated from an existing image. This is the current stage.

## Generated-from-data geometry

A geometry image/SVG generated from the same data model used by the runtime. This is the target stage.
