# WC2026 gameboard visual defaults

The gameboard outline defaults were tuned in the developer frame and promoted from copied live properties.

Current defaults:

- opacity: `0.52`
- line color: `rgba(255, 255, 255, 0.98)`
- line width: `1.5`
- glow: `0.05`

These defaults keep the board visible but not overpowering over the pub background.
