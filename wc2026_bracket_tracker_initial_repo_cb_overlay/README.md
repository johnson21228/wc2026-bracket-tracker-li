# WC2026 Bracket Tracker Initial Repo Capture Back Overlay

This overlay is intended to create or initialize a new Workbench repo from the WB template.

Recommended repo name:

```text
wc2026-bracket-tracker-li
```

The overlay captures:

- the Workbench purpose
- LI rules
- poster-derived source evidence
- static HTML release posture
- two-game bracket pool model
- storage model
- starter JSON files
- initial cards and prompts
- Capture Back note

The Workbench can be built from any II reasoner. The static HTML is the user-facing surface. The Workbench is the durable source of truth.
