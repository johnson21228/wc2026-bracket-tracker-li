# Capture Back — Gameboard Group Button Rail

This capture defines and implements the group button rail.

The bottom gameboard rail gives every group a persistent entry point to the shared group panel. Each tile shows the group label over a 2×2 square flag grid for the four teams in that group.

The rail is an inspection surface. It does not pick teams, clear picks, or mutate bracket advancement. It routes through the existing group panel path and uses only local checked-in group/team data.
