# Capture Back — Game 1 R16 Choice Menu Rules

This capture-back records the rule boundary for Game 1 R16 winner menus.

The R16 menu is sourced from bracket adjacency: each R16 slot receives two R32 source slots, and its menu offers exactly the teams assigned to those source slots. The menu becomes available only when both source teams exist.

No runtime behavior is changed by this overlay. It establishes LI, documentation, prompt, and card authority so the following runtime patch can be narrow and testable.
