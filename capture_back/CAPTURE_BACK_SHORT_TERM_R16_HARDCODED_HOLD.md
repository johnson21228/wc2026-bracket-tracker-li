# Capture Back: Short-Term R16 Hardcoded Hold

## Summary

Adds a temporary bridge that writes the selected first-left R32 winner into `L-R16-01` so the team can be held and rendered in a Round of 16 cell while the full knockout assignment path is repaired.

## Boundary

This is explicitly short-term. It proves that R16 storage and rendering can hold a selected R32 winner.
