# Start-Today Workbench Prompt Lesson

## Lesson

The template should preserve a reusable prompt that generates the current best-practice Start-Today Workbench guide with copy and an infographic.

## Captured insight

"Stage 0" can be useful internally, but the stronger teaching frame is:

```text
Start-Today Workbench
```

This avoids making the first surface feel primitive or obsolete.

## Why it matters

Start-Today Workbench is already a real Workbench pattern:

- one chat window;
- two roles;
- local repo memory;
- terminal Capture Back;
- review surface;
- human commit;
- history-based re-entry.

## Product implication

Future generated Workbenches should be able to explain how to start using Workbench immediately, before any dedicated product surface exists.

## One-line formulation

Start-Today Workbench is not a lesser demo. It is the first practical Workbench surface.
