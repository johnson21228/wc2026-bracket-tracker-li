# Optional Continuity Note Example

## Goal

Capture extra handoff context for a meaningful Workbench loop when commit history and repo history are not enough.

## Example

```text
Goal:
Simplify the Workbench loop explanation for first-time collaborators.

Change:
Added high-level loop LI and a simple infographic prompt.

Validation:
make verify passed.
make pack passed.
Committed and repacked.

Handoff:
Use the simple loop for first impressions. Use the detailed loop for operators.
```

## Reminder

This note is optional.

The Workbench loop does not require a card or note.
