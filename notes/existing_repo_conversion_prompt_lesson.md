# Existing Repo Conversion Prompt Lesson

## Lesson

The template already had a prompt for converting an existing repo, but it was too thin for the real user workflow.

A user may upload an existing repo pack and ask to transfer the Workbench structure into it. The correct response is not to clone the template wholesale. The correct response is to inspect the pack, preserve the repo's purpose, and add the minimum Workbench LI layer needed for continuity and Capture Back.

## Product implication

Workbench adoption needs a clear bridge from ordinary repos to Workbench repos.

That bridge should be a first-class template prompt, because it is one of the most likely entry paths for real use.
