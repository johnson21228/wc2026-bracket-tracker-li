# Capture Back macOS Review Surface Image Lesson

## Lesson

The generated Capture Back on macOS infographic successfully intersects the earlier Stage 0 visual pattern with the newer Capture Back review-surface LI.

## What changed

The visual no longer leads with stage language.

It teaches the operational protocol:

- one chat, two roles;
- download overlay to `~/Downloads`;
- apply from repo root in Terminal;
- run verification and pack;
- open review files;
- paste stdout back into chat;
- commit only after review;
- use Workbench history for re-entry.

## Why it matters

The image makes a complex human-in-the-loop procedure teachable at a glance.

It turns Capture Back from an abstract repo update into a visible custody loop.

## Principle

A Workbench visual is strongest when it makes the human review boundary visible.
