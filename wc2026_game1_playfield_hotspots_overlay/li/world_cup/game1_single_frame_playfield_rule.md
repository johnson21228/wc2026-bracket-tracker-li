# Game 1 Single-Frame Playfield Rule

Game 1 asks the player to predict which team fills each official Round-of-32 qualification slot.

Use a single-frame playfield image as the visible tournament surface. Transparent hit-test rectangles sit over each R32 slot.

Interaction:

```text
hover slot -> show official slot rule
click slot -> show allowed teams
select team -> flag appears in slot
```
